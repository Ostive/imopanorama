# Images ImoPanorama

Ce dossier est organise pour remplacer progressivement les placeholders par de vraies photos Madagascar sans changer le code.

## Dossiers

- `brand/`
  - `logo.png` : logo principal du site.
- `hero/`
  - `home-agent.png` : visuel principal de la page d'accueil.
- `auth/`
  - `auth-illustration.png` : visuel des pages login/register/reset.
- `social/`
  - `og-image.png` : image partage social/SEO.
- `properties/`
  - `property-placeholder.jpg` : fallback pour les biens sans photo.
  - `featured/property-1.jpg` a `property-6.jpg` : emplacements pour biens reels en avant.
- `batipanorama/`
  - `project-placeholder.jpg` : fallback projet construction.
  - `hero/construction-madagascar.jpg` : hero BatiPanorama.
  - `projects/project-1.jpg` a `project-5.jpg` : photos de projets construction.
- `news/`
  - `news-1.jpg` a `news-3.jpg` : images d'articles en fallback.
- `seo/cities/`
  - `antananarivo.jpg`
  - `tamatave.jpg`
  - `majunga.jpg`
  - `fianarantsoa.jpg`
  - `nosy-be.jpg`
  - `diego-suarez.jpg`
- `markers/`
  - icones de carte.
- `placeholders/`
  - anciens placeholders conserves pour compatibilite.

## Photos reelles a fournir en priorite

1. `hero/home-agent.png` : photo ou montage premium pour l'accueil.
2. `properties/featured/property-1.jpg` a `property-6.jpg` : vrais biens Madagascar.
3. `seo/cities/*.jpg` : vues reelles des villes/quartiers.
4. `batipanorama/projects/project-1.jpg` a `project-5.jpg` : chantiers, maisons terminees, plans ou equipes.
5. `news/news-1.jpg` a `news-3.jpg` : photos editorialisees pour les articles.
6. `social/og-image.png` : image lisible en partage Facebook/WhatsApp/LinkedIn.

## Recommandations

- Hero desktop : 1600x1000 minimum, JPG/WebP, sujet bien centre.
- Cards biens : 1200x800, ratio 3:2 ou 4:3.
- Villes SEO : 1600x900, photo lumineuse et reconnaissable.
- Logo : PNG transparent.
- Nommer les fichiers en minuscules, sans espaces, avec tirets.
- Garder les anciens noms si possible : remplacer le fichier suffit, pas besoin de modifier le code.
